# RB-004.6 Privileged Access Assessment

## Objective

Assess privileged account exposure, administrative activity, and potential privilege escalation following account compromise.

---

## Status

Live

---

## Inputs

- Authentication logs
- Administrative activity logs
- Role assignment telemetry
- Endpoint telemetry
- Session activity logs

---

## Tools

- Identity provider
- SIEM
- PAM platform
- EDR platform
- Cloud audit logs

---

## Steps

### 1. Identify Privileged Access Exposure

Determine:
- Administrative roles assigned
- Active privileged sessions
- Privileged application access
- Shared administrative accounts

---

### 2. Review Administrative Activity

Assess:
- Role assignment changes
- Policy modifications
- Access control changes
- Security configuration changes
- Administrative API activity

---

### 3. Identify Privilege Escalation Indicators

Look for:
- New privileged accounts
- Group membership changes
- Conditional access modifications
- MFA policy changes
- Service principal abuse

---

### 4. Review Lateral Movement Activity

Identify:
- Remote administrative access
- Cross-system authentication
- Privileged session hopping
- Administrative tool usage

---

### 5. Restrict or Remove Privileged Access

Perform:
- Session revocation
- Role removal
- Credential rotation
- Temporary administrative restrictions

---

### 6. Validate IAM Integrity

Confirm:
- No rogue administrators remain
- Access policies intact
- Administrative logging operational
- Conditional access functioning correctly

---

## Decision Points

| Condition | Action |
|----------|--------|
| Privilege escalation confirmed | Escalate to PB-009 |
| Widespread administrative activity observed | Expand investigation scope |
| IAM policy tampering identified | Immediate executive escalation |

---

## Output

- Privileged access assessment
- Administrative activity timeline
- Role exposure inventory
- IAM integrity assessment

---

## Common Failure Modes

- Failing to review inherited privilege
- Ignoring service account exposure
- Leaving privileged sessions active

---

## Automation Hooks

- Auto-role inventory collection
- Auto-privilege anomaly detection
- Auto-session revocation
