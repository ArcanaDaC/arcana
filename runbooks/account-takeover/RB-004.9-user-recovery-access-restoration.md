# RB-004.9 User Recovery & Access Restoration

## Objective

Restore secure access for impacted users following account compromise while ensuring attacker access has been fully removed.

---

## Status

Live

---

## Inputs

- Containment status
- Session revocation status
- MFA reset status
- Identity integrity assessment
- Endpoint assessment findings

---

## Tools

- Identity provider
- MFA platform
- Endpoint management platform
- SIEM
- Service desk tooling

---

## Steps

### 1. Validate Containment Completion

Confirm:
- Sessions revoked
- Tokens invalidated
- OAuth abuse addressed
- Rogue devices removed
- Credentials reset

---

### 2. Validate Endpoint Integrity

Review:
- Malware investigation findings
- Endpoint telemetry
- Browser security posture
- Device compliance status

---

### 3. Reset Credentials Securely

Require:
- Strong password creation
- MFA enforcement
- Identity verification
- Password reuse prevention

---

### 4. Re-Enroll MFA

Validate:
- Trusted MFA methods
- Approved devices
- Recovery methods
- Conditional access enforcement

---

### 5. Restore User Access

Re-enable:
- Identity provider access
- SaaS access
- VPN access
- Email access
- Administrative access (if applicable)

---

### 6. Monitor for Re-Compromise

Monitor:
- New suspicious logins
- Session anomalies
- OAuth abuse
- Authentication failures
- Suspicious endpoint activity

---

## Decision Points

| Condition | Action |
|----------|--------|
| Reinfection or reuse observed | Re-isolate immediately |
| Endpoint integrity uncertain | Delay restoration |
| Repeated suspicious logins observed | Expand investigation scope |

---

## Output

- User recovery status
- Access restoration timeline
- MFA validation status
- Residual risk assessment

---

## Common Failure Modes

- Restoring access before endpoint validation
- Reusing compromised MFA methods
- Ignoring browser token persistence

---

## Automation Hooks

- Auto-password reset workflows
- Auto-MFA enrollment validation
- Auto-risk monitoring
- Auto-session anomaly detection
