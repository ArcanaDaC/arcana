# RB-005.3 Data Staging Investigation

## Objective

Identify pre-exfiltration staging behaviour, archive preparation activity, and temporary storage locations used to facilitate data theft.

---

## Status

Live

---

## Inputs

- Endpoint telemetry
- File access logs
- EDR telemetry
- User activity logs
- Network telemetry

---

## Tools

- EDR platform
- SIEM
- File integrity monitoring
- Endpoint forensic tooling

---

## Steps

### 1. Identify Archive Creation Activity

Review:
- ZIP/RAR/7z creation
- Archive encryption usage
- Large file creation
- Compression utilities
- Temporary archive directories

---

### 2. Review File Collection Activity

Identify:
- Bulk file access
- Sensitive directory access
- File copying activity
- Network share access
- Unusual search behaviour

---

### 3. Assess Staging Locations

Review:
- Temporary directories
- User profile locations
- Cloud sync folders
- Mounted storage
- Shared directories

---

### 4. Identify Automation or Scripting

Review:
- PowerShell scripts
- Batch files
- Python scripts
- Scheduled tasks
- Data collection tooling

---

### 5. Correlate with Transfer Activity

Determine:
- Which staged files were transferred
- Transfer timing correlation
- Archive-to-transfer relationships
- Compression-to-exfiltration timing

---

### 6. Assess Scope Expansion

Identify:
- Additional staging systems
- Shared staging infrastructure
- Multi-user involvement
- Historical staging activity

---

## Decision Points

| Condition | Action |
|----------|--------|
| Active staging ongoing | Escalate containment |
| Automated collection identified | Expand hunting scope |
| Sensitive archives identified | Escalate severity |

---

## Output

- Staging activity timeline
- Archive inventory
- Staging location assessment
- Correlated transfer findings

---

## Common Failure Modes

- Focusing only on transfers
- Ignoring temporary directories
- Missing staged-but-untransferred data

---

## Automation Hooks

- Auto-archive detection
- Auto-large file monitoring
- Auto-sensitive directory monitoring
- Auto-staging correlation
